## 💻 Code Preview

### 🔹 Class & Object

```cpp
class Student {
public:
    string name;
    int age;
};
```

---

### 🔹 Encapsulation

```cpp
class Bank {
private:
    int balance;

public:
    void setBalance(int b) {
        balance = b;
    }

    int getBalance() {
        return balance;
    }
};
```

---

### 🔹 Inheritance

```cpp
class Animal {
public:
    void eat() {
        cout << "Eating...";
    }
};

class Dog : public Animal {};
```

---

### 🔹 Polymorphism

```cpp
class Base {
public:
    virtual void show() {
        cout << "Base";
    }
};

class Derived : public Base {
public:
    void show() {
        cout << "Derived";
    }
};
```
